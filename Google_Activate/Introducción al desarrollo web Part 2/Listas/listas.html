  • Description List

    It is used for glossaries, with "dl", it is defined as a description list, with "dt" the term to be defined is placed, with "dd" the definition is placed. For example:

    Song
    Sung musical piece.
    Literary composition, usually in verse, set to music to be sung.
    Dance
    Action of dancing.
